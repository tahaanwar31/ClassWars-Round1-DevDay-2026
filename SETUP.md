# ClassWars Setup Guide

## Project Structure

```
ClassWars-Round1-DevDay-2026/
├── frontend/          # React app (Game + Admin)
└── backend/           # NestJS API
```

## Setup Instructions

### 1. Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

✅ Frontend runs on: **http://localhost:3000**

### 2. Backend Setup

```bash
cd backend
npm install
npm run start:dev
```

✅ Backend runs on: **http://localhost:3002**

## Access Application

- **Game**: http://localhost:3000
- **Admin**: http://localhost:3000/admin/login
  - Email: `admin@classwars.com`
  - Password: `admin123`

## That's It!

Dono folders independent hain. Har ek mein:
1. `cd` karo
2. `npm install` karo
3. `npm run dev` karo

Done! 🎉
